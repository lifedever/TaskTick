Features功能特色

Everything you need.
Nothing you don't.

你需要的全都有，
不需要的一个没有。

Built for developers who want scheduled tasks without the complexity of system daemons.

为不想折腾系统守护进程的开发者而生。

Menu Bar Resident

菜单栏常驻

Lives in your menu bar, always running in the background. Close the main window — TaskTick keeps working quietly.

常驻菜单栏，后台静默运行。关闭主窗口也不影响任务执行。

Cron & Interval Scheduling

Cron 表达式 & 固定间隔

Full cron expression support with visual presets and human-readable preview. Or set simple fixed intervals.

完整的 Cron 表达式支持，内置可视化预设和可读预览。也可设置固定间隔。

Script Execution

脚本执行

Write scripts inline or select local files — .sh, .py, .rb, .js and more. Choose your shell, set working directory.

支持内联脚本或选择本地文件 — .sh、.py、.rb、.js 等。自由选择 Shell 和工作目录。

Execution Logs

执行日志

Every run is logged — stdout, stderr, exit codes, and duration. Filter by status, browse history with full context.

每次执行自动记录 — 标准输出、错误输出、退出码、耗时。支持状态筛选。

System Notifications

系统通知

Get macOS native notifications on task success or failure. Configure per task — only hear about what matters.

任务成功或失败时推送 macOS 原生通知。支持按任务独立配置。

Bilingual: EN & 中文

中英双语

Full English and Simplified Chinese interface. Switch instantly or follow your system preference.

完整的中英文界面，App 内一键切换语言，或跟随系统偏好。

Auto Updates

自动更新

Automatically checks GitHub Releases for new versions. Get notified and download updates with one click.

自动检查 GitHub Releases 上的新版本，发现更新时提醒并一键下载。

macOS 26 Liquid Glass

macOS 26 液态玻璃

Supports the new liquid glass visual effects on macOS 26 with graceful fallbacks for older systems.

支持 macOS 26 全新液态玻璃视觉特效，旧系统优雅降级。

Preview预览

Your scripts, your schedule

你的脚本，你的时间表

TaskTick supports any script that runs in a shell. Write inline or point to a file — it handles execution, logging, and notifications automatically.

TaskTick 支持任何能在 Shell 中运行的脚本。内联编写或指向文件 — 自动执行、记录和通知。

  • Bash, Python, Ruby, Node.js, and moreBash、Python、Ruby、Node.js 等
  • Capture stdout, stderr, and exit codes捕获标准输出、错误输出和退出码
  • Custom working directory per task每个任务可自定义工作目录
  • Notification on success or failure成功或失败时通知

# Task: backup-db
# Schedule: 0 2 * * * (daily at 2:00 AM)
# Shell: /bin/bash
